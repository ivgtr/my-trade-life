## 遊び方

初期資金100万円を元手に、3分間のデイトレードを繰り返し、資産を増やしていきます。毎朝届くニュースや市場の雰囲気を読み取りながら、株の売買タイミングを見極めましょう。

1. タイトル画面からニューゲームを開始
2. カレンダー画面で営業日を選択して出勤
3. モーニング画面でその日のニュースと市場予測を確認
4. セッション画面でリアルタイムに動くチャートを見ながら売買
5. セッション終了時、ポジションを持っていれば「全決済」か「翌日に持ち越し」を選択
6. レポート画面でその日の成績を振り返り
7. 月末には月次レポートで収支を確認

これを繰り返してビリオネア（資産10億円）を目指しましょう！

## 勝利と敗北

- 資産が10億円に到達するとビリオネアエンド。達成後はエンドレスモードで続行も可能
- 残高が0円以下になり、保有ポジションもない状態でゲームオーバー

## 操作方法

セッション画面では成行注文でトレードを行います。

- LONG（買い）/ SHORT（空売り）ボタンで注文
- 株数は直接入力のほか、+1 / +10 / +100 / MAX ボタンで調整
- 各ポジションにストップロス（SL）/ テイクプロフィット（TP）を設定可能
- ポジションごとの個別決済、または全決済が可能
- ヘッダーから 1x / 2x の速度切り替えが可能
- 持ち越したポジションは翌日の始値で自動決済される

## ゲームの特徴

- リアルタイムに変動する株価チャートを見ながらトレード
- ニュース速報が株価に影響を与える
- 市場全体の景気が周期的に変化する
- トレードを重ねると経験値が貯まりレベルアップ
- 月次・年次レポートで自分の成績を振り返れる
- セーブ/ロード、データのインポート・エクスポートに対応
- BGMと効果音でゲーム体験を演出

## ローカルで遊ぶ

```bash
pnpm install
pnpm dev
```

## 注意事項

本ゲームは完全なフィクションです。架空の株価データを使用しており、実際の投資とは一切関係ありません。
